# Roadmap phát triển website Portfolio (phía website)

## Phase 1: Khởi tạo & Cơ bản

- Khởi tạo project Next.js, cấu hình TypeScript, Tailwind CSS, ESLint.
- Thiết lập cấu trúc thư mục chuẩn, tạo layout cơ bản (Header, Footer, Theme/Language Switcher).
- Xây dựng trang chủ với các section: Hero, About, Featured Projects, Tech Stack.
- Responsive, tối ưu UI/UX cơ bản.

## Phase 2: Trang Projects & Blog

- Thiết kế và xây dựng trang Projects (danh sách, chi tiết dự án, phân loại nếu cần).
- Thiết kế và xây dựng hệ thống Blog (hiển thị danh sách, chi tiết bài viết).
- Tích hợp markdown cho nội dung blog.
- Thêm phân trang, tìm kiếm cơ bản cho blog và projects.
- Tối ưu SEO cho các trang chính, blog và projects.

## Phase 3: Đa ngôn ngữ & Theme nâng cao

- Tích hợp hệ thống đa ngôn ngữ (i18n), chuyển đổi ngôn ngữ động.
- Hoàn thiện chuyển đổi theme (dark/light), lưu trạng thái theme.
- Tối ưu accessibility (a11y).

## Phase 4: Hiệu ứng & Tối ưu

- Thêm animation cho các section, hiệu ứng chuyển trang.
- Tối ưu hiệu suất tải trang (image, code splitting, lazy load).
- Kiểm thử, fix bug UI/UX.

## Phase 5: Hoàn thiện & Đóng gói

- Viết tài liệu hướng dẫn sử dụng, đóng gói sản phẩm.
- Refactor code, chuẩn hóa cấu trúc, bổ sung test nếu cần.
- Chuẩn bị cho phase phát triển phía admin (dashboard quản trị).

---

> Lưu ý: Roadmap này chỉ áp dụng cho phía website (end-user). Phía admin sẽ có roadmap riêng.
